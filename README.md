# Mutual Fund Analysis using Python

## Project Overview
This project analyzes mutual fund data using Python to understand fund performance, risk levels, and return patterns. It helps investors make better investment decisions using data-driven insights.

## Objectives
- Analyze mutual fund performance across categories
- Compare returns of different funds
- Study risk vs return patterns
- Identify best performing mutual funds
- Understand investment trends

## Tools & Libraries Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset Description
The dataset includes mutual fund details such as:
- Fund name
- Category
- Returns (1Y, 3Y, 5Y)
- Risk level
- Expense ratio
- Fund size

## Key Insights
- Equity mutual funds generally provide higher returns over long term
- Balanced funds offer stable but moderate returns
- Higher risk funds tend to give higher returns
- Expense ratio affects overall fund performance

## Conclusion
This project provides insights into mutual fund performance and helps investors choose better investment options based on risk and return analysis.
